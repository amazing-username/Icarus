# Icarus

Icarus is a music streaming API that interacts with [Mear](https://github.com/amazing-username/mear).

### Prerequisites
* dotnet-core-dsk
* dotnet-core-runtime
* dotnet-core-host

## Built With

* C#
* .NET Core
* .NET Web RESTful API

## Versioning

Currently under development. No version has been released
